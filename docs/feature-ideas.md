# 機能アイデアメモ

## 目的

このメモは、仕事中に手元で使える小さな便利機能の候補を整理するものです。

現在はポモロードと UUID 作成が主な機能として存在しています。今後も、重い業務管理機能より、開いてすぐ使える小道具を少しずつ増やしていく方針が扱いやすいと考えられます。

## 優先して検討したい機能

| 優先度 | 機能 | 用途 |
| --- | --- | --- |
| 高 | JSON 整形・圧縮 | API レスポンスや設定値を貼り付けて、読みやすく整える |
| 高 | 日時・タイムスタンプ変換 | Unix 秒、ISO 日時、JST/UTC を相互に確認する |
| 高 | Base64 encode/decode | JWT や簡単なペイロードの確認、エンコード済み文字列の展開に使う |
| 高 | URL encode/decode | クエリ文字列、リダイレクト URL、コールバック URL の確認に使う |
| 高 | JWT デコーダー | JWT のヘッダーとペイロードを表示し、内容を素早く確認する |

## 次点の候補

| 優先度 | 機能 | 用途 |
| --- | --- | --- |
| 中 | 正規表現テスター | 入力文字列に対するマッチ結果を確認する |
| 中 | 文字数・バイト数カウンター | Slack 文面、DB 制限、API 制限などに収まるか確認する |
| 中 | ハッシュ生成 | 文字列から SHA-256 や MD5 などを生成する |
| 中 | diff ビューア | 2 つのテキスト、設定値、レスポンスの差分を確認する |
| 中 | QR コード生成 | URL や検証用リンクをスマートフォンで開きやすくする |

## 余裕があれば検討する機能

| 優先度 | 機能 | 用途 |
| --- | --- | --- |
| 低 | カラーピッカー・色変換 | HEX、RGB、HSL を相互変換する |
| 低 | 簡易メモ・今日の作業ログ | ポモロードと組み合わせて、その日の作業メモを残す |

## 実装順の候補

最初に追加するなら、次の順番がよさそうです。

1. JSON 整形・圧縮
2. 日時・タイムスタンプ変換
3. Base64 / URL encode・decode
4. JWT デコーダー
5. 文字数・バイト数カウンター

これらは実装が比較的軽く、仕事中の利用頻度も高いと考えられます。まずはクリップボードや外部ファイル連携を前提にせず、テキストを貼り付けて結果を確認できる形から始めると、機能を小さく保ちやすくなります。
